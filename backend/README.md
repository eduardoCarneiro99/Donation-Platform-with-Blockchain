# Seminars
 Platform for blockchain (ethereum) based donations to charity.
 
## Pre-requisites
- Ganache installed
- Node installed
 
## Build
### If there were changes to the contracts, inside the src folder run:
```
truffle compile
```
### and while ganache is running:
```
truffle migrate
```

## To build:
```
npm run build
```

## How to run
### Development with change detection:
```
npm serve
```

### To start:
```
npm run start
```
